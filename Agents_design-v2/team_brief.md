Team Brief: Design & UX (SIMPLES)

Objetivo

Fazer o app parecer um assistente simples: enviar arquivo → processar → baixar resultado.

Regra máxima

Se não ajuda o usuário AGORA, não aparece.

Ordem de trabalho

UX Strategist define fluxo e tela 1

Layout Architect organiza hierarquia e respiro

UI Designer define visual mínimo

UX Writer simplifica textos

Acessibilidade valida legibilidade e mobile

QA testa “5 segundos” e corta ruído

Definição de sucesso

Tela inicial: 1 ação principal (upload + processar)

Pós-processo: resultado + downloads

Zero “painel técnico” na abertura
